## GitHub Projects Blog

안녕하세요! 이 블로그는 **GitHub 인기 프로젝트**를 소개하는 자동화 블로그입니다.

### 목표

- 🔥 **인기 프로젝트**: GitHub에서 추세 중인 프로젝트 수집
- 📝 **한국어 리뷰**: AI로 자동으로 상세 리뷰 작성
- 🎯 **SEO 최적화**: 검색 엔진 최적화
- 💰 **수익화**: 개발자 친화적 제휴 링크 제공

### 자동화 기술

이 블로그는 다음 기술로 자동화됩니다:

1. **GitHub API**: 인기 프로젝트 트렌딩 수집
2. **OpenAI GPT-4o-mini**: 한국어 콘텐츠 생성
3. **Hugo**: 정적 사이트 빌드
4. **GitHub Actions**: 자동 배포

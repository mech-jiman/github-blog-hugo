# Next.js 완벽 가이드: React 프레임워크의 모든 것

Vercel에서 개발한 Next.js는 현대 웹 애플리케이션 개발을 혁신하는 React 프레임워크입니다. 13만 이상의 GitHub 스타와 3만 이상의 포크를 기록한 Next.js는 서버 사이드 렌더링(SSR), 정적 사이트 생성(SSG), 증분적 정적 재생성(ISR) 등 다양한 렌더링 전략을 단일 프레임워크에서 구현할 수 있게 해줍니다.

## 핵심 기능

1. **서버 사이드 렌더링(SSR)**  
   요청 시 서버에서 페이지를 렌더링하여 SEO 친화적이고 빠른 첫 로딩을 제공합니다.

2. **정적 사이트 생성(SSG)**  
   빌드 시점에 페이지를 사전 생성하여 CDN에서 즉시 제공 가능한 초고속 정적 사이트를 구축합니다.

3. **증분적 정적 재생성(ISR)**  
   정적 페이지를 백그라운드에서 자동 재생성하여 최신 콘텐츠와 성능을 동시에 유지합니다.

4. **파일 시스템 기반 라우팅**  
   `pages` 디렉토리의 파일 구조가 자동으로 라우팅으로 변환되며 동적 라우트도 간편하게 구현 가능합니다.

5. **API 라우트**  
   백엔드 API를 프론트엔드와 동일한 프로젝트 내에서 빌드할 수 있는 서버리스 함수 지원입니다.
